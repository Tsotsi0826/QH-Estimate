Getting Started

To start using the Construction Estimator:

  1. Create a new client or open an existing client using the buttons in the sidebar.
  2. Select a module from the sidebar list to begin estimating.
  3. Add new custom modules or category headers using the "Add New Module" button.
  4. Drag and drop modules in the sidebar to reorder them or place them under headers.
